What is Secure Code?

Secure code is the practice of developing programs that prevents accidential security incidents and safeguards programs from potential vulnerabilities. Upon inspection of the application's code it will assess potential weaknesses including:

  • SQL Injection
  • Cross-site Scripting (XSS)
  • Insecure Authentication Mechanisms
  • Common Security Flaws
